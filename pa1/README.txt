1. When first logging in, you will be prompted for a password.
2. Go to Account Settings.
3. In the CS255 keys table, enter a group name and a key generated by a friend, or enter a group name and generate a new key.  You should not attempt to use a key that wasn't generated by our system.
4. Share the generated key with your friends.
5. Post encrypted messages in the group!

Notes:
After 3 failed attempts/cancels, keys will not be loaded, and a default key of all 0's will be used.  If a key is in an invalid format, then the default key will also be used.
